# CalendarHub — Event Scheduler with Calendar

Monthly calendar view with event scheduling and reminders.

**Live:** https://calendarhub-indol.vercel.app

## Features
- Monthly calendar view
- Create and manage events
- Event reminders
- Color-coded categories

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite

## Getting Started
```bash
npm install
npm run dev
```
